🚀 Getting Started
1. Clone the repo
git clone https://github.com/GitforAnkit16/React-Component.git
cd React-Component
2. Install dependencies
npm install
3. Run the dev server
npm run dev
App will be available at 👉 http://localhost:5173
4. Run Storybook
npm run storybook
Storybook will be available at 👉 http://localhost:6006
